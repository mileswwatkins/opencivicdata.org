How to format your Spreadsheet

The spreadsheet accepts data in a very simple format, which is aimed to cover input of the most basic information about a particular Jurisdiction.

Basic Format

You can include any subset of fields below, with the exception of Name, which is required for all sheets. In addition, you may add a label to any of the headers, which is denoted by surounding the label with parentheses after thee Header key. For instance, both Address and Address (home) are valid headers.

Valid Keys

Name
Name of the person that the data in the row corresponds to.
District
District that the person represents. This may be empty for people that don't have a district, such as a Mayor.
Position
Position that the person holds, something like "Councilmember", or "Mayor".
Image
Headshot of the person. This is totally optional.
Phone
Phone number for the person.
Address
Postal address where one can send mail to this person.
Email
Email address where one can send mail to this person.
Fax
Facsimile address where one can send documents to this person.
Cell
Cell phone number of the person.
Website
Official website of this person.
Homepage
Personal or official homepage of this person.
Twitter
Twitter account of this person. Full URL.
Facebook
Facebook account of this person. Full URL.
Blog
Blog website of this person.
Webform
Contact webform of this person. This is not the ideal method to collect contact information for people.
Employer
Person's employer.
Source
Where you found the data you used for this person.
